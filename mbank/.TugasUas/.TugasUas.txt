
Bank
Buatlah program manajemen bank.
User dapat membuka rekening baru, menabung, dan menarik uangnya.
Tidak lupa fungsi perhitungan bunga.

Syarat fitur yang harus ada:
1.	Login user, daftar user baru. Pada saat daftar, user diharuskan menabung minimal 500ribu.
2.	Menabung, menarik uang.
3.	Perhitungan bunga berdasarkan metode “Saldo Rata-Rata”.
4.	User dapat melihat saldo mereka dan melihat log transaksi mereka selama 30 hari terakhir.
5.	Saran field tabel user: ID User, Username, Alamat, Pekerjaan.
6.	Saran field tabel transaksi: ID Transaksi, ID User, Tanggal, Tipe Transaksi, Jumlah Transaksi, Saldo. Tipe transaksi berupa debet/kredit.
